Full setup guide:
  1. Go to developer.spotify.com/dashboard
  2. Click Create app (any name/description)
  3. In Redirect URIs field, paste the green URI shown above
  4. Under Which API/SDKs are you planning to use? tick Web API
  5. Click Save, then copy your Client ID from the app overview
  6. Paste it above and click Connect
